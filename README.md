# Painel Administrativo - Doce Lagom

Painel administrativo para gerenciamento de pedidos e estoque da Doce Lagom.

## Credenciais de Acesso
- **Usuário:** admin
- **Senha:** lagom123

## Funcionalidades

### 1. Dashboard
- Resumo de vendas
- Pedidos pendentes
- Ticket médio
- Estatísticas em tempo real

### 2. Gestão de Pedidos
- Visualização de todos os pedidos
- Atualização de status (Pendente → Confirmado → Entregue)
- Detalhes completos dos pedidos

### 3. Controle de Estoque
- Monitoramento de produtos
- Reposição de estoque
- Alertas para baixo estoque

### 4. Produtos Mais Vendidos
- Ranking dos produtos mais populares
- Quantidade de vendas por produto

### 5. Ações Rápidas
- Repor todo o estoque
- Exportar dados (backup)
- Limpar todos os dados (reset)

## Tecnologias Utilizadas
- HTML5
- CSS3 (Flexbox, Grid, Variáveis CSS)
- JavaScript Vanilla (ES6+)
- localStorage para persistência de dados
- Font Awesome para ícones
- Google Fonts (Montserrat, Playfair Display)
